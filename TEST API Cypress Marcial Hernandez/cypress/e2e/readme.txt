Instrucciones de Ejecución:

Abre Cypress: Asegúrate de tener Cypress instalado. Puedes abrirlo ejecutando el comando npx cypress open en tu terminal desde la carpeta del proyecto.

Selecciona el Archivo de Prueba: En la ventana de Cypress, selecciona el archivo de prueba que contiene el código para las pruebas de la PetStore API.

Ejecuta las Pruebas: Haz clic en el archivo de prueba para ejecutar todas las pruebas definidas en él. Las pruebas se ejecutarán automáticamente.

Verificación de Resultados: Observa la ejecución de las pruebas en la ventana de Cypress. Si todas las pruebas se ejecutan con éxito, verás un mensaje indicando que todas las pruebas pasaron.
